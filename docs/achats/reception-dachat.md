# Réception d'achat
_Les réceptions d'achat permettent l'ajout d'articles inclus dans une commande d'achat à un inventaire._

---

Il est possible de créer une réception d'achat à partir d'une commande d'achat, ou de manière manuelle.

## Créer une réception d'achat à partir d'une commande d'achat

1. Dans la commande d'achat que vous désirez réceptionner, appuyez sur le + à droite de **Réception achat** dans la section de connexions rapides.


![image (3) (1) (1).png](../../static/assets/image%20%283%29%20%281%29%20%281%29.png)

2. Inscrivez la quantié accepté pour les différents articles de la commande d'achat.
3. Sélectionnez l'entrepot dans lequel vous désirez stocker les articles reçus. (optionel)
4. Appuyez sur Sauvegarder, puis sur Soumettre afin de compléter la réception de la commande d'achat.

## Créer une réception d'achat manuellement

1. Dans l'écran de commande d'achats, appuyez sur `Ajouter réception d'achat`.
2. Sélectionnez le fournisseur de la réception d'achat
3. Dans la section **Articles**, ajoutez les articles à réceptionner en appuyant sur **Ajout multiple**.
4. Appuyez sur `Sauvegarder`, puis sur `Soumettre` pour enregistrer la réception d'achat.
