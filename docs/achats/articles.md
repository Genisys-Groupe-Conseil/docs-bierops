# Articles
_Les articles sont la base de toute recette, inventaire, commande et réception._

---

Suite à une importation de bière, les articles inclus dans la recette seront automatiquement ajoutés dans la liste d'articles de BierOps. Par contre, certaines configurations sont nécessaires avant de pouvoir créer une commande d'achat.

## Configuration

Dans la section Achat de l'article, inscrivez les informations suivantes :

* Fournisseur par défaut.
* Unité de mesure d'achat.
* Quantité de commande minimum.
* Stock de sécurité.
* Code d'article du fournisseur.
