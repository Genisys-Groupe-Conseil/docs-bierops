# Commande d'achat
_La commande d'achat est un document délivré par un acheteur à un fournisseur, indiquant les types, les quantités et les prix convenus pour les produits demandés._

---

Dans BierOps, il est possible de créer une commande d'achat de manière automatique en utilisant l'option de [planification d'achat](planification-dachat.md), par contre vous pouvez également créer une commande d'achat manuellement. Les commandes d'achat sont liées aux réceptions d'achat et aux factures d'achat, ce qui vous permet de garder une vue d'ensemble de vos transactions d'achat.

## Créer une commande d'achat

![Créer une commande d'achat](../../static/assets/image%20%282%29%20%281%29%20%281%29%20%281%29%20%281%29%20%281%29.png)

1. Dans l'écran de commande d'achat, appuyez sur Ajouter commande d'achat.
2. Sélectionnez un fournisseur
3. Dans le champ **Requis le**, inscrivez la date requise pour la commande (optionel).
4. Ajouter les articles de la commande dans la section **Articles**.
   1. Sélectionnez le code d'article.
   2. Inscrivez la date requise dans le champ **Requis le**.
   3. Inscrivez la quantité ainsi que l'unité de mesure (Udm) de l'article.
5. Appuyez sur Sauvegarder pour enregister la commande d'achat.
