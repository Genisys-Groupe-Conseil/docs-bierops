# Planification d'achat
_Le menu de planification d'achat vous permet de calculer les requis matériels afin de générer automatiquement les commandes d'achats des différents fournisseurs._

---

BierOps propose les achats selon une méthode "juste à temps", afin de ne pas encombrer votre entrepot de matiere requise plus tard.

## Créer une planification d'achat


![image (1) (1) (1) (1) (1) (1).png](../../static/assets/image%20%281%29%20%281%29%20%281%29%20%281%29%20%281%29%20%281%29.png)

1. Dans l'écran de planification d'achat, appuyez sur `Ajouter Planification d'achat`.
2. Inscrivez une description pour la planification d'achat (optionel).
3. Utilisez la section **Filtres** afin de limiter les articles proposés selon vos critères.
4. La section **Articles Requis** affiche tous les articles requis selon les lots de bière planifiés en production. Sélectionnez les articles que vous désirez inclure dans la planification d'achat, puis appuyez sur `Planifier les achats pour les articles sélectionnés`.
5. Vous trouverez les résultats de la planification dans la section **Articles à acheter**. Celle-ci présente un tableau des articles indquant la quantité à acheter pour chacun d'eux.
6. Appuyez sur `Sauvegarder`, puis sur `Soumettre` dans le coin supérieur droit pour soumettre la planification d'achat.

Si la quantité à acheter d'un article affiche 0, cela signifie que la quantité en inventaire réponds au besoins de production.
