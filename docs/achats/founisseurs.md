# Fournisseurs
_Le fournisseur est au centre de vos transactions d'achat. Il est lié dans les demandes de soumissions, les commandes d'achat, les réceptions et les paiements._

---

Grâce à la fiche du fournisseur, vous pouvez suivre efficacement des éléments essentiels tels que :

* les adresses multiple et les contacts du fournisseur
* les comptes débiteurs
* limite de crédit et période de crédit

## Créer un fournisseur

1. Appuyez sur `Ajouter Fournisseur` dans le coin supérieur droit dans l'écran fournisseurs.
2. Inscrivez le nom du fournisseurs que vous désirez ajouter.
3. Sélectionnez le groupe de fournisseurs associé à celui-ci.
4. Sélectionnez le type de fournisseur (Société dans la majorité des cas).
5. Sélectionnez ou créez une [liste de prix pour le fournisseur](liste-de-prix-fournisseurs.md).
6. Inscrivez les détails du contact principal ainsi que l'adresse principale (optionnel).
7. Appuyez sur `Sauvegarder`.

Pour une configuration avancé d'un nouveau fournisseur appuyez sur "Ouvrir le formulaire complet".
